# Project instructions

The authoritative source for LLM/assistant guidance on this repo is
`inst/site/llms.txt`. Read it for the package overview, layout, build
system, conventions, and rules for modifying code.

`.github/copilot-instructions.md` is a mirror of the same content, kept
solely so that GitHub Copilot's context loader finds it.

Do not duplicate long-form guidance across files -- edit
`inst/site/llms.txt` and treat the others as thin aliases.
